# Google Space Clone 🚀

A creative redesign of the Google homepage with a futuristic space theme using HTML and CSS.

## Features

* Google-inspired homepage layout
* Space-themed gradient background
* Starry background effect
* Glassmorphism search card
* Responsive search bar
* Interactive buttons with hover effects
* Modern UI design

## Technologies Used

* HTML5
* CSS3

## Project Structure

```text
Google_Clone/
│
├── index.html
├── style.css
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Explore the Google Space Edition interface.

Through this project, I learned:

* HTML page structure
* CSS Flexbox alignment
* CSS gradients and visual effects
* Glassmorphism design
* Hover animations and transitions
* Responsive design basics
